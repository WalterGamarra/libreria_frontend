# 📚 Librería Online — Frontend

Interfaz web para una tienda de libros online. Incluye una vista pública del catálogo y un panel de administración con autenticación para usuarios registrados.

---

## 🛠️ Tecnologías

- **HTML5**
- **CSS3**
- **JavaScript**
- **Netlify** — deploy en producción

---

## 📌 Funcionalidades

- 🏪 Catálogo público de libros sin necesidad de login
- 🔐 Login con autenticación JWT
- 📖 Agregar y consultar libros según rol del usuario
- 🚪 Cierre de sesión
